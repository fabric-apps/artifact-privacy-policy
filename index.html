Privacy Policy for ARTIFACT

Last updated: June 2026

ARTIFACT does not collect, transmit, or store any personal data, browsing history, or user content.

What ARTIFACT does:

  • Injects a formatting toolbar into claude.ai for improved prompt composition.
  • Stores only your preference for whether the toolbar is enabled or disabled. This is saved locally using Chrome's storage.local API and never transmitted anywhere.
  • Does not access, read, or store any conversation content or any data from claude.ai beyond what is necessary to format the text you manually type into the input field.

Data sharing: ARTIFACT does not share any data with third parties. There are no analytics trackers, no backend servers (in Phase 1), and no external API calls.

Permissions:

  • storage: Used solely to remember your toolbar preference.
  • activeTab: Used to interact with the claude.ai tab when you click the toolbar buttons.
  • host_permissions for https://claude.ai/*: Required to inject the toolbar into the page.

Contact: If you have questions, open an issue on our GitHub repository.
