ConText for LLMs - Privacy Policy

Last updated: June 2026

ConText does not collect, transmit, or store any personal data, browsing history, or user content.

What ConText does:

  • Injects a formatting toolbar into chatgpt.com, claude.ai, gemini.google.com for improved prompt composition.
  • Stores only your preference for whether the toolbar is enabled or disabled. This is saved locally using Chrome's storage.local API and never transmitted anywhere.
  • Does not access, read, or store any conversation content or any data from chatgpt.com, claude.ai, gemini.google.com beyond what is necessary to format the text you manually type into the input field.

Data sharing: ConText does not share any data with third parties. There are no analytics trackers, no backend servers (in Phase 1), and no external API calls.

Permissions:

  • storage: Used solely to remember your toolbar preference.
  • activeTab: Used to interact with chatgpt.com, claude.ai, gemini.google.com tab when you click the toolbar buttons.
  • host_permissions for chatgpt.com, claude.ai, gemini.google.com: Required to inject the toolbar into the page.

Contact: If you have questions, open an issue on our GitHub repository.
